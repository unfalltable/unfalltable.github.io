Handhat

简介

  • 一个灵活、快速、可扩展的以太坊智能合约开发环境

  • 本地开发编译部署测试

  • 具有丰富的Debug工具

  • 组成部分

    • Runner：运行环境
    • Network：模拟EVM网络
    • vscode：插件

  • 目录结构

    • contracts：合约文件

    • scripts：脚本文件

    • tasks：任务

    • test：测试

    • hardhat.config.ts：Hardhat配置文件

命令

命令 作用 参数 备注
npx hardhat init 初始化项目
npx hardhat compile 编译合约 在配置文件中配置版本
npx hardhat node 启动本地网络
npx hardhat run scripts/deploy.ts –network 网络类型
